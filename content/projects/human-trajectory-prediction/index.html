Key Highlights

  • Real-world data was collected in a 26m × 20m factory environment using an advanced LiDAR system.
  • A long short-term memory (LSTM) neural network was used to predict the workstation a person is heading toward, up to 3 seconds in advance.
  • To overcome limited real-world data, two data augmentation techniques were used: TimeGAN (AI-generated trajectories) and a traditional Ornstein-Uhlenbeck process (statistical simulations).
  • Combining synthetic and real data improved accuracy from 87.5% to 95.6% for predictions 3 seconds ahead.

Why It Matters

In smart factories, predicting where workers are going lets machines react in real time—either by preparing tools or adjusting speed to avoid collisions. This makes factories safer, more efficient, and better aligned with human needs.

Challenges Tackled

  • Data scarcity: Human movement data is hard to collect. The team addressed this with synthetic data generation and transfer learning.
  • Real-time accuracy: The model can correctly predict a person’s destination up to 3 seconds before they arrive with >95% accuracy.
  • Privacy: The LiDAR setup anonymises data, respecting participant privacy.

Technologies Used

  • LiDAR-based human tracking
  • Stacked LSTM network for prediction
  • TimeGAN and OU process for synthetic data generation
